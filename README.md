# Mahakali 1000 Names Website

This site displays the 1000 names of Mahakali (Mahakali Sahasranama) with short meanings and elaborations, styled and structured similarly to the Kalabhairava 1000 names website.

- Data source: `../mahakali1000nameswebsite/mahakali_sahasranama_meanings.json`
- Main page: `index.html`
- Styling: `styles.css`
- Logic: `app.js`

## Usage
Open `index.html` in a browser to view the names.
